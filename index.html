Nesting

nesting artinya sarang: sangat simple, yaitu menyimpang selector di selector yang lain .

Nesting digunakan untuk membuat CSS Combinators, CSS Pseudo-class ataupun CSS Pseudo-element. Dengan CSS standar, elemen herarki diakses melalui karakter spasi.

Latihan variable

variable menggunakan scss/sass hampir sama dengan html, menggunakan dollar $ sebagai tandanya.

sifat sifat variable pada sass

map pada variable

map adalah salah satu model penulisan / tempat menyimpan lebih dari satu warna
cara penulisannya adalah dengan membuat variable, lalu di dalam kurung di kasih value dan di pisah dengan koma, untuk memanggi warnanya dengan "map-get" : $variabel,'nama value nya'.
